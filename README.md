# hoardCTI/template

The `template` repo is the base template for all hoardCTI repositories — providing standardised rulesets, tests, and licensing out of the box.

## Why this repo exists

Hoard CTI is an enterprise-ready, open-source threat intelligence feed. Given the use case and the organisations relying on it, every repo needs production-ready rules and tests to keep users safe, compliant, and operating within a consistent standard.

This repo defines that baseline, providing:

- Version control rulesets
- Access control and licensing
- Required tests and approval workflows
